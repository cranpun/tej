1. make api on google app script (reference)[https://qiita.com/cranpun/items/7ed08f855359c9c28a33]

## how to use

### en -> ja

```commandline
node /path/to/main.js en sentence
```

### ja -> en

```commandline
node /path/to/main.js ja sentence
```
